# 資料說明

## 1. 資料來源

本研究使用 TEJ 台股資料，包含復權後 OHLCV 欄位；部分實驗加入三大法人籌碼資料。

由於 TEJ 與籌碼資料皆涉及授權，本 repo 不提供原始行情資料與完整 parquet 檔案，只提供程式碼、實驗設定、樣本輸出與結果摘要。

---

## 2. 主要欄位

| 欄位 | 說明 |
|---|---|
| stock_id | 股票代號 |
| date | 交易日期 |
| open | 開盤價 |
| high | 最高價 |
| low | 最低價 |
| close | 收盤價 |
| volume | 成交量 |
| ret | 報酬率 |
| chip features | 三大法人相關籌碼特徵，僅用於籌碼版本實驗 |

---

## 3. 資料期間

| 資料類型 | 可用期間 | 說明 |
|---|---|---|
| TEJ OHLCV | 2000-01-01 至 2025-12-31 | 價格影像模型主要資料來源 |
| 三大法人籌碼資料 | 2008-01-01 至 2025-12-31 | 因資料取得期間限制，含籌碼實驗從 2008 年開始 |
| 樣本外測試 | 2018-01-01 至 2025-12-31 | 目前主要測試期間 |

---

## 4. 樣本建立方式

每一筆模型樣本包含：

1. 股票代號。
2. 影像結束日期。
3. 過去 window 日的市場資料。
4. 由 OHLC、報酬率、成交量或籌碼特徵生成的價格影像。
5. 未來 horizon 日報酬方向標籤。
6. 樣本外預測分數。

---

## 5. 缺值處理

若影像視窗內包含 NaN、資料長度不足，或無法計算未來報酬標籤，該樣本不加入訓練、驗證或測試索引。

含籌碼資料的實驗若遇到籌碼欄位缺值，也會在建立樣本索引或影像通道時排除。

---

## 6. 資料限制

1. 本 repo 不提供 TEJ 原始資料。
2. 本 repo 不提供完整 parquet 檔案。
3. 三大法人籌碼資料可用期間從 2008 年開始，因此含籌碼版本不能從 2000 年開始完整訓練。
4. 本 repo 僅提供研究流程、程式碼、設定檔、樣本圖與結果摘要。
