# 資料說明

## 1. 資料來源

本研究使用 TEJ 台股資料，包含復權後 OHLCV 欄位。

由於 TEJ 為授權資料，本 repo 不提供原始行情資料，只提供程式碼、實驗設定、樣本輸出與結果摘要。

---

## 2. 主要欄位

| 欄位 | 說明 |
|---|---|
| stock_id | 股票代號 |
| date | 交易日期 |
| open | 開盤價 |
| high | 最高價 |
| low | 最低價 |
| close | 收盤價 |
| volume | 成交量 |
| ret | 報酬率 |

---

## 3. 樣本建立方式

每一筆模型樣本包含：

1. 股票代號。
2. 影像結束日期。
3. 過去 window 日的市場資料。
4. 由 OHLC 或其他通道生成的價格影像。
5. 未來 horizon 日報酬方向標籤。
6. 樣本外預測分數。

---

## 4. 樣本期間

| 區間 | 日期 |
|---|---|
| 訓練集 | 2008-01-01 至 2014-12-31 |
| 驗證集 | 2015-01-01 至 2017-12-31 |
| 測試集 | 2018-01-01 至 2025-12-31 |

---

## 5. 缺值處理

若影像視窗內包含 NaN、資料長度不足，或無法計算未來報酬標籤，該樣本不加入訓練、驗證或測試索引。

---

## 6. 資料限制

1. 本 repo 不提供 TEJ 原始資料。
2. 本 repo 不提供完整 parquet 檔案。
3. 本 repo 僅提供研究流程、程式碼、設定檔、樣本圖與結果摘要。
