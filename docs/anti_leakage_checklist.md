# Anti-Leakage Checklist

本文件用來檢查研究流程是否存在前視偏誤或資料外洩。

## 1. 時間順序

- [ ] 影像只使用 t 日以前資料。
- [ ] label 使用 t+1 到 t+h 的未來報酬。
- [ ] 測試期間不參與模型訓練。
- [ ] 測試期間不參與模型選擇。
- [ ] 不使用未來成交量、未來價格、未來籌碼。

## 2. 標準化與統計量

- [ ] 圖像標準化參數只用訓練集估計。
- [ ] 驗證集與測試集不得回頭影響訓練特徵。
- [ ] 缺值處理不使用未來資訊。

## 3. 股票池

- [ ] 股票池過濾不使用未來資訊。
- [ ] 不使用未來是否下市作為當期過濾條件。
- [ ] 每個再平衡日只使用當時已知資料。

## 4. 回測

- [ ] 交易成本有明確設定。
- [ ] 持有期與再平衡規則明確。
- [ ] D10-D1 權重計算不使用未來報酬。
- [ ] 統計檢定在樣本外資料上進行。

## 5. 過度擬合

- [ ] 不只呈現最好看的 seed。
- [ ] 保留所有實驗摘要。
- [ ] 檢查不同 seed 的結果。
- [ ] 檢查不同測試期間的結果。
